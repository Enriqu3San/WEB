Definition Lists

A definition list in HTML is used to display a list of terms and their corresponding descriptions. It is typically used to represent glossary items, key-value pairs, or any other structured content that involves definitions or explanations.

The list is created using the < dl> (definition list) element, with each term enclosed in a < dt> (definition term) tag and each description or definition in a < dd > (definition description) tag.
